Tableau Dashboards

Insight map: unveiling COVID-19 dynamics

In this project I conducted exploratory data analysis using SQL queries. This allowed me to extract valuable insights such as the infection rates, mortality rates, and the regional trends for such rates and turn these insights into valuable visualizations that formed a basis for the dashboard. I then leveraged Tableau's capabilities in dashboard creation to create an interactive dashboard using these visualizations that are very user-friendly. I then implemented geographical mapping into the display to allow for easier visualization of the regional trends as well as pick-and-choose regions to look at.
