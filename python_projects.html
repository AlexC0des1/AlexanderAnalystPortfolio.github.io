Data Science Job EDA

In this project I utilized Python's pandas library to analyze data science job listings so that I could gain knowledge on what it would take to start a new career in the Data Science/Analytics field. I then implemented regex techniques to help me examine the key qualities that the job descriptions had for the career. I then visualized this data using seaborn and used a wordcloud to compile a plan to be successful in the career.

Keeping Photos safe one pixel at a time

In this project I designed an application that used the Advanced Encryption Standard (AES) method of encryption to secure photos from my local files. I also looked into other popular encryption methods that could have been used to encrypt the photos. I then developed a method for which I can send these encrypted photos over email safely. I lastly created a method for which the photos could be decrypted with the right decryption key.
