Power BI Dashboards

Data professional insights: unveiling trends and patterns

In this project I utilized gathered responses, from a survey performed by Alex Freberg, from professionals in the data domain in Microsoft Excel for further analysis. I then leveraged Microsoft Power BI to clean, preprocess, and perform exploratory data analysis on the data received from the survey. This allowed me to gather insight into these responses by creating visualizations with this data. These visualizations were then converted into a dynamic and user-friendly dashboard that showcases key findings from the data and shows a comprehensive overview of the survey results.

What is the best football team?

In this project I utilized gathered international football, commonly known as soccer in the United States, from 1872-2023 to create an interactive dashboard that can showcase interesting statistics of the games played. I used Microsoft Power BI to clean the data then I performed exploratory data analysis on the dataset. This allowed me to create clean visualizations of the winning ratios by creating filters to look at the year, the games won which can be filtered by home and away games, and finally the type of tournament the game was played in. Using this information I created ideas on which team can be considered the best.
